# DATA 612 – Project 1: Global Baseline Predictors and RMSE

**Author:** Kevin Martin

## Overview

This project explores the fundamentals of recommender systems by building a simple baseline recommendation model using a toy movie ratings dataset.

The project demonstrates how to:

- Create a user-item matrix
- Split ratings into training and test datasets
- Calculate a global average rating
- Calculate user and item biases
- Build baseline predictors
- Evaluate model performance using RMSE (Root Mean Squared Error)

## Files

- `DATA612_Project_1_Global_Baseline_RMSE.ipynb` - Main project notebook

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Project Objective

The goal of this project is to understand how simple recommender systems make predictions before moving on to more advanced techniques such as collaborative filtering, matrix factorization, and hybrid recommendation systems.
